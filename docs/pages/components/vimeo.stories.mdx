## Help

<Story name="help" />

To embed a Vimeo video using the `<Vimeo />` component grab the `vimeoId` from the URL.

```
https://vimeo.com/38445453
```

The `vimeoId` is this bit.

```
38445453
```

Then use it like this.

```jsx
<Vimeo vimeoId="38445453" />
```
