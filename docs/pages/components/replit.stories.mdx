## Help

To embed an Snack using the `<Replit />` component grab last pieces of the query string of the url

```
"https://replit.com/@ritza/demo-embed
```

The repl are the last bits

```
@ritza/demo-embed
```

Then use it in the component like this.

```jsx
<Replit repl="@ritza/demo-embed" />
```

Learn more: https://docs.replit.com/hosting/embedding-repls
