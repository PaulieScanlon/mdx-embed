# LinkedInBadge

Display a LinkedIn Badge by including the component in your `.mdx`. The `username` prop is required.

_⚠️ If you're seeing ERR_ABORTED 999 in your console its because LinkedIn block requests when the rate limit is exceeded by any single IP address_
