## Help

<Story name="help" />

To embed a YouTube video using the `<YouTube />` component grab the `youTubeId` from the URL.

```
https://www.youtube.com/watch?v=WUgvvPRH7Oc
```

The `youTubeId` is this bit.

```
WUgvvPRH7Oc
```

Then use it like this.

```jsx
<YouTube youTubeId="WUgvvPRH7Oc" />
```

To embed a YouTube playlist using the `<YouTube />` component grab the `youTubePlaylistId` from the URL.

```
https://www.youtube.com/watch?v=C8M-JhhYaBs&list=PL5SvzogSTpeH1Szqw4tPi9ZfgXDbY8GU-
```

The `youTubePlaylistId` is this bit.

```
PL5SvzogSTpeH1Szqw4tPi9ZfgXDbY8GU-
```

Then use it like this.

```jsx
<YouTube youTubePlaylistId="PL5SvzogSTpeH1Szqw4tPi9ZfgXDbY8GU-" />
```
