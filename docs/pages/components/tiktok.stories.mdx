## Help

<Story name="help" />

To embed a TikTok player using the `<TikTok />` component grab the `username`, `video` and `id` from the TikTok URL.

```
https://www.tiktok.com/@scout2015/video/6718335390845095173#
```

The `username`, `video` and `id` are these bits.

```
@scout2015/video/6718335390845095173
```

Then use it like this.

```jsx
<TikTok tikTokId="@scout2015/video/6718335390845095173" />
```
