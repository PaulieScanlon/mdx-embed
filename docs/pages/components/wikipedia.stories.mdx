## Help

<Story name="help" />

To embed a Wikipedia post using the `<Wikipedia />` component grab the `wikipediaLink` from the URL.

```
https://en.wikipedia.org/wiki/Tim_Berners-Lee
```

The `wikipediaLink` is this bit.

```
Tim_Berners-Lee
```

Then use it like this.

```jsx
<Wikipedia wikipediaLink="Tim_Berners-Lee" />
```
