## Help

<Story name="help" />

To embed a Airtable base using the `<AirtableBase />` component grab the `airtableBaseId` and the embed code.

It should be everything between `https://airtable.com/embed/` and `?`.

```html
<iframe
  class="airtable-embed"
  src="https://airtable.com/embed/shrVJbC9yrwSS18XZ?backgroundColor=green&layout=card&viewControls=on"
  frameborder="0"
  onmousewheel=""
  width="100%"
  height="533"
  style="background: transparent; border: 1px solid #ccc;"
></iframe>
```

The `airtableBaseId` is this bit.

```
shrVJbC9yrwSS18XZ
```

Then use it like this.

```jsx
<AirtableBase airtableBaseId="shrVJbC9yrwSS18XZ" />
```
