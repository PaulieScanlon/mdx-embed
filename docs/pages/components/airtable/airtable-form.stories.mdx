## Help

<Story name="help" />

To embed a Airtable base using the `<AirtableForm />` component grab the `airtableFormId` and the embed code.

It should be everything between `https://airtable.com/embed/` and `?`.

```html
<iframe
  class="airtable-embed"
  src="https://airtable.com/embed/shr0dHXnGG9IA2xDc?backgroundColor=green"
  frameborder="0"
  width="100%"
  height="533"
  style="background: transparent; border: 1px solid #ccc;"
></iframe>
```

The `airtableFormId` is this bit.

```
shr0dHXnGG9IA2xDc
```

Then use it like this.

```jsx
<AirtableForm airtableFormId="shr0dHXnGG9IA2xDc" />
```
