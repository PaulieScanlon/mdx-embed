## Help

<Story name="help" />

To embed a Wistia video using the `<Wistia />` component, once you've uploaded video in Wistia, grab the `videoId` from the URL.

```
https://user-name.wistia.com/medias/yplqa7pg87
```

The `videoId` is this bit.

```
yplqa7pg87
```

Then use it like this.

```jsx
<Wistia videoId="yplqa7pg87" />
```
