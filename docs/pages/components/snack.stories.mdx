## Help

<Story name="help" />

To embed a Snack using the `<Snack />` component grab the last pieces of the query string from the url

```
https://snack.expo.io/@pauliescanlon/test
```

The `snackId` is made up of the username and the part of the url

```
@pauliescanlon/test
```

Then use it like this.

```jsx
<Snack snackId="@pauliescanlon/test" />
```
