# Snack

Display an embed [Snack](https://snack.expo.io) into your mdx file.
The `snackId` is required.

## Help

<Story name="help" />

To embed an Snack using the `<Snack />` component grab last pieces of the query string of the url

```
https://snack.expo.io/@matiasfh/animated-image
```

The snackId are the last bits

```
@matiasfh/animated-image
```

Then use it in the component like this.

```jsx
<Snack snackId="@matiasfh/animated-image" />
```
