## Help

<Story name="help" />

To embed a Cinnamon video using the `<Cinnamon />` component grab the `cinnamonId` from the URL.

```
https://cinnamon.video/watch?v=389807855663842407
```

The `cinnamonId` is this bit.

```
389807855663842407
```

Then use it like this.

```jsx
<Cinnamon cinnamonId="389807855663842407" />
```
