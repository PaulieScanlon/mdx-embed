## Help

<Story name="help" />

To embed a Buzzsprout podcast episode player using the `<Buzzsprout />` component grab the `buzzsproutId` and episodeId from the URL.

It should be everything after https://www.buzzsprout.com

```
https://www.buzzsprout.com/1081172/4728410-react-hooks
```

The `buzzsproutId` is this bit.

```
1081172/4728410-react-hooks
```

Then use it like this.

```jsx
<Buzzsprout buzzsproutId="1081172/4728410-react-hooks" />
```
